# College Website Project – PHP & MySQL (Project 7)

## Description
College website project using **PHP**, **MySQL**, and **CSS**. Features student registration, viewing records, deleting entries, and session management.

## Features
- Student Registration Form
- View all students
- Delete student records
- Session management (login/logout)
- Styled with CSS

## Project Structure
college-website-project7-php-mysql/
├── index.php
├── login.php
├── dashboard.php
├── logout.php
├── create_db.php
├── create_table.php
├── insert.php
├── view.php
├── style.css
└── connect.php


## Requirements
- XAMPP (Apache + MySQL)
- PHP 7.x or higher
- Browser (Chrome/Firefox)
- IDE: VS Code / Notepad++

## How to Run Locally
1. Copy the project folder to `C:\xampp\htdocs\`.
2. Start **Apache** and **MySQL** in XAMPP.
3. Open browser → `http://localhost/college-website-project7-php-mysql/index.php`.
4. Create database & table if not already created.
5. Add, view, or delete student records.

## Notes
- GitHub cannot run PHP/MySQL code directly.
- Use a PHP hosting service (e.g., 000webhost) to run online.
